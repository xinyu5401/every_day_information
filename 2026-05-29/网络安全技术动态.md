# 网络安全技术动态（2026-05-29）

## 1. Jenkins 2026-05-27 安全公告：多个插件修复权限、凭据和 XSS 风险

**热度判断：** Jenkins 官方安全公告发布，CI/CD 属于高价值攻击面；插件漏洞常被用于凭据窃取、流水线篡改和供应链攻击入口。

**技术细节：** 公告覆盖多个 Jenkins 插件问题，风险类型包括缺少权限校验、敏感信息暴露、跨站脚本和 CSRF/表单校验不足等。Jenkins 插件通常运行在控制器权限上下文中，一旦漏洞允许低权限用户读取配置、枚举凭据或触发构建操作，可能扩大到源码、制品仓库、云凭据和部署环境。

**影响与处置：** 受影响组织应核对插件清单和版本，优先升级公告列出的修复版本；对公网 Jenkins、共享 CI 和多租户流水线，应同步审计最近 30 天插件配置变更、异常构建触发、凭据读取和 webhook 调用。无法立即升级时，应限制低权限用户访问相关任务和管理端点。

**信息来源：**
- https://www.jenkins.io/security/advisory/2026-05-27/
- https://www.jenkins.io/security/advisories/

## 2. Fortinet FortiClient EMS 远程代码执行风险持续被关注

**热度判断：** FortiClient EMS 属于企业终端安全管理基础设施，漏洞利用价值高；近期安全社区继续关注 Fortinet PSIRT 公告与 CVE 修复节奏。

**技术细节：** FortiClient EMS 管理端通常具备终端策略下发、软件分发和资产管理能力。若管理服务存在 SQL 注入、命令注入、反序列化或认证绕过类问题，攻击者可从管理平面扩展到终端执行和横向移动。需要重点核对 Fortinet PSIRT 中对应 CVE、受影响版本、是否需要认证、是否存在公开 PoC/在野利用，以及补丁是否改变默认暴露端口或认证流程。

**影响与处置：** 将 FortiClient EMS 管理界面从互联网隔离，仅允许运维网段访问；升级到官方修复版本；检查管理端日志中的异常 API 调用、批量终端策略变更、未知安装包下发和管理员账户新增。漏洞编号、CVSS 和利用状态以 Fortinet PSIRT 原文为准。

**信息来源：**
- https://www.fortiguard.com/psirt
- https://www.fortiguard.com/psirt?date=2026-05

## 3. 国内漏洞与威胁情报源例行复核：CNVD、CNCERT、FreeBuf

**热度判断：** CNVD/CNCERT/FreeBuf 是国内漏洞通报、攻击活动分析和厂商响应的重要信源；本期用于复核是否存在需要进入日报的重大国内事件。

**技术细节：** 本期重点检索高危漏洞、在野利用、供应链攻击、云服务配置风险和攻防工具公开。对 CNVD 编号类事件，纳入标准为同时具备对象、版本、触发条件、危害等级和修复建议；对媒体转载或泛新闻，未展开为独立事件。

**影响与处置：** 建议持续监测 CNVD 漏洞库、CNCERT 风险通报和 FreeBuf 技术文章；若后续出现明确 CNVD 编号、PoC 或厂商补丁，应按资产暴露面和可利用性重新排序。

**信息来源：**
- https://www.cnvd.org.cn/
- https://www.cert.org.cn/
- https://www.freebuf.com/
