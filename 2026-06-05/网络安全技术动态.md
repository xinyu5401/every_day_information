## CISA KEV 到期窗口

### CVE-2025-48595：Android Framework 整数溢出本地提权今日到期

**入选理由**：CISA KEV 记录 `dateAdded=2026-06-02`、`dueDate=2026-06-05`，命中今日处置窗口；Android 官方 2026-06 安全公告将其列入 Framework 漏洞。

**技术细节**：CISA 描述该漏洞为 Android Framework integer overflow，可导致本地权限提升并可能实现代码执行。该类漏洞通常要求攻击者先在设备上获得本地执行入口，再利用 Framework 权限边界缺陷扩大权限。CISA 已确认存在在野利用证据，但未标注勒索软件使用。

**影响与处置**：优先覆盖 Android 企业终端、BYOD 管理设备、移动办公设备和高权限账号常用设备。处置动作包括推送 2026-06 Android 安全补丁、核查厂商 OTA 覆盖率、对无法升级设备限制企业资源访问，并关注 EDR/MDM 中异常本地提权与未知应用安装痕迹。

**信息来源**：
- https://www.cisa.gov/sites/default/files/feeds/known_exploited_vulnerabilities.json
- https://source.android.com/docs/security/bulletin/2026/2026-06-01
- https://nvd.nist.gov/vuln/detail/CVE-2025-48595

### CVE-2022-0492：Linux cgroups v1 release_agent 提权今日到期

**入选理由**：CISA KEV 记录 `dateAdded=2026-06-02`、`dueDate=2026-06-05`，命中今日处置窗口；该漏洞历史上与容器逃逸、低权限到 root 提权链条相关。

**技术细节**：CISA 描述为 Linux Kernel improper authentication，可通过 cgroups v1 `release_agent` 特性触发权限提升。风险点集中在仍启用 cgroup v1、容器隔离配置宽松、宿主机内核补丁滞后的环境。

**影响与处置**：优先排查 Kubernetes/容器宿主机、CI runner、共享开发机和多租户 Linux 主机。建议确认内核补丁状态，禁用不必要的 cgroup v1 能力，限制容器 `CAP_SYS_ADMIN`、特权容器和宿主机路径挂载，并在 EDR/SIEM 中检索异常写入 `release_agent`、挂载 cgroup 和容器内提权行为。

**信息来源**：
- https://www.cisa.gov/sites/default/files/feeds/known_exploited_vulnerabilities.json
- https://nvd.nist.gov/vuln/detail/CVE-2022-0492

### CVE-2026-45247：Mirasvit Full Page Cache Warmer 反序列化 RCE 明日到期

**入选理由**：CISA KEV 记录 `dateAdded=2026-06-03`、`dueDate=2026-06-06`，命中未来 24 小时窗口；受影响对象是 Magento 生态的缓存预热插件，属于互联网暴露电商资产常见攻击面。

**技术细节**：CISA 描述该漏洞为不可信数据反序列化，未认证攻击者可通过构造 `CacheWarmer` cookie 中的序列化 PHP 对象触发远程代码执行。若站点存在可用 gadget chain，风险会从应用层对象注入升级为 Web 服务器权限下命令执行。

**影响与处置**：优先排查公网 Magento/Adobe Commerce 站点、第三方插件清单和 WAF 命中日志。处置动作包括升级或禁用 Mirasvit Full Page Cache Warmer、清理可疑 Web shell、检索异常 cookie、PHP 反序列化错误和未知后台管理员账号，并对无法确认插件版本的站点先做外网访问限制。

**信息来源**：
- https://www.cisa.gov/sites/default/files/feeds/known_exploited_vulnerabilities.json
- https://sansec.io/research/mirasvit-cache-warmer-object-injection
- https://nvd.nist.gov/vuln/detail/CVE-2026-45247
