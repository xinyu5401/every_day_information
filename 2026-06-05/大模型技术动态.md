## NVIDIA Nemotron 3 Ultra 550B：开放权重长上下文推理模型发布

**入选理由**：NVIDIA 在 Hugging Face 发布 `NVIDIA-Nemotron-3-Ultra-550B-A55B-NVFP4`，模型卡标注发布日期为 2026-06-04；Firecrawl 厂商发布雷达同时命中 NVIDIA Build 模型卡。

**技术细节**：该模型总参数 550B、激活参数 55B，采用 LatentMoE 架构，包含交错的 Mamba-2、MoE 与部分 Attention 层；训练使用 NVFP4 预训练配方，并保留部分关键层为 BF16/MXFP8 以提升稳定性。模型面向 agentic workflow、长上下文分析、工具调用、多语言推理和代码/数学/科学任务；模型卡给出最高 1M token 上下文能力，部署示例默认 256k，可通过配置提升到 1,048,576。评测中 SWE-Bench Verified 为 71.9（BF16）/69.7（NVFP4），SWE-Bench Multilingual 为 67.7/65.8。训练数据披露包括约 20T token 预训练语料、后训练合成数据、异步 GRPO 多环境强化学习，以及部分开放的数据集合；许可证为 OpenMDW-1.1。

**影响与处置**：这是 550B 级开放权重推理模型进入 NVFP4 与 1M 上下文部署路线的实质更新，适合纳入内部长上下文 RAG、代码修复和多工具 Agent 的离线评测。上线前应重点复测中文、代码生成、安全拒答、工具调用参数污染和长上下文检索稳定性；模型卡建议部署硬件为 4x B200 单节点，实际成本和吞吐需按业务上下文长度重新测算。公开模型卡未标注按量 API 单价。

- https://build.nvidia.com/nvidia/nemotron-3-ultra-550b-a55b/modelcard

## OpenAI GPT-Rosalind：生物学推理模型能力更新

**入选理由**：OpenAI 于 2026-06-03 发布 GPT-Rosalind 能力更新，属于过去 36 小时头部厂商补漏热点；该更新包含明确 benchmark、推理成本和可用性信息。

**技术细节**：OpenAI 称新版本在 GPQA 上从 51.6% 提升到 70.5%，同时在 SuperGPQA 与主观专家评估中超过前代；推理成本约降低 10 倍，延迟下降约 4 倍。产品定位仍是生物学、实验设计和科学推理场景，而不是通用聊天模型；官方页面未公开完整模型权重、上下文长度、API 单价或系统卡细节。

**影响与处置**：科研与生物医药团队可把 GPT-Rosalind 纳入实验规划、文献推理和假设生成评测，但不应跳过 wet-lab、合规和双用途风险审查。涉及生物安全的内部调用应记录 prompt、工具、外部数据库和输出用途，避免把模型输出直接作为实验执行指令。

**信息来源**：
- https://openai.com/index/introducing-new-capabilities-to-gpt-rosalind/

## OpenAI ChatGPT Memory “Dreaming”：后台反思式记忆更新机制上线

**入选理由**：OpenAI 于 2026-06-04 发布 ChatGPT 记忆系统更新，主题不是新模型权重，而是对长期用户记忆的更新、压缩和质量控制机制的产品级改动。

**技术细节**：新系统引入“dreaming”式后台反思流程，在用户会话后异步整理可长期保存的偏好与上下文。OpenAI 表示该流程为每次记忆更新投入约 5 倍计算量，用更少但更有价值的记忆替代频繁追加；系统会周期性重估旧记忆的相关性，减少过时上下文干扰。

**影响与处置**：企业或高敏用户需要重新审视 ChatGPT 记忆策略：明确哪些信息允许进入长期记忆，哪些需要关闭记忆或使用临时聊天；对涉及客户资料、代码、密钥、病例和内部流程的场景，应把“长期记忆可见性、删除流程、审计记录”纳入使用规范。

**信息来源**：
- https://openai.com/index/chatgpt-memory-dreaming/
