# 网络安全技术动态（2026-06-09）

## CISA KEV 到期窗口

### 1. CVE-2026-45321：TanStack npm 供应链投毒进入 KEV 处置窗口

- 入选理由：CISA KEV 已收录，`Date Added=2026-05-27`，`Due Date=2026-06-10`，命中今日至未来 24 小时处置窗口；且为已确认在野利用的开发者工具供应链事件。
- 技术细节：NVD 记录显示，攻击者在 `2026-05-11 19:20-19:26 UTC` 间通过 TanStack 的 GitHub Actions OIDC trusted publishing 绑定发布了 `42` 个 `@tanstack/*` 包的 `84` 个恶意版本。利用链是 `pull_request_target` 误用 + fork/base 边界上的 Actions cache poisoning + 运行时内存提取 OIDC token，随后直接向 npm 发布恶意包。
- 影响与处置：该事件本质上是 CI/CD 与包仓库联动的供应链攻击，不应仅按“恶意 npm 包”处理。应立即核查过去 30 天内的安装记录、锁文件、缓存镜像、私有代理仓库和 CI runner 缓存，并核对是否命中过去 24 小时内仍在安装旧缓存。若命中，按主机失陷处理并轮换云凭据、GitHub、npm、SSH、Vault 等密钥。
- 信息来源：
  - https://www.cisa.gov/known-exploited-vulnerabilities-catalog
  - https://nvd.nist.gov/vuln/detail/CVE-2026-45321
  - https://tanstack.com/blog/npm-supply-chain-compromise-postmortem
  - https://github.com/TanStack/router/security/advisories/GHSA-g7cv-rxg3-hmpx

### 2. CVE-2026-48027：Nx Console 恶意扩展进入 KEV 处置窗口

- 入选理由：CISA KEV 已收录，`Date Added=2026-05-27`，`Due Date=2026-06-10`，命中今日至未来 24 小时处置窗口；同时该事件覆盖 `Visual Studio Marketplace`、`Open VSX` 与 VS Code 系生态，属于高优先级开发者终端供应链风险。
- 技术细节：NVD 与 Nx 官方复盘显示，恶意 `Nx Console 18.95.0` 于 `2026-05-18 12:30 UTC` 发布到 Visual Studio Marketplace，并在 `12:33 UTC` 上架 Open VSX；Visual Studio Marketplace 约 `18` 分钟后下架，Open VSX 约 `36` 分钟后下架。攻击上游来自 TanStack 恶意包导致的开发者机 `gh` 凭据泄露，下游表现为恶意 VS Code 扩展在激活时落地持久化文件、窃取本地与云凭据，并尝试通过 HTTPS 与 DNS 通道外传。
- 影响与处置：凡在暴露窗口内启用 Nx Console 自动更新的 VS Code 或 Cursor 终端，应按潜在失陷处理；重点检查扩展版本、`~/.local/share/kitty/cat.py`、`/var/tmp/.gh_update_state`、Windows `%TEMP%\\kitty-*` 等 IOC，并轮换 GitHub、npm、云平台、1Password/CLI 会话和 SSH 凭据。对企业侧，应把扩展自动更新与开发者 OAuth token 持久化纳入基线控制。
- 信息来源：
  - https://www.cisa.gov/known-exploited-vulnerabilities-catalog
  - https://nvd.nist.gov/vuln/detail/CVE-2026-48027
  - https://nx.dev/blog/nx-console-v18-95-0-postmortem
  - https://github.com/nrwl/nx-console/security/advisories/GHSA-c9j4-9m59-847w
