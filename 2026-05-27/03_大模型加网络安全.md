## Agent 攻击与自动化渗透

### Evo-Attacker：长期工具攻击进入强化学习建模

- **时间**：arXiv recent，发布时间待核实。
- **要点**：Evo-Attacker 将针对 LLM 多智能体系统的长期工具攻击建模为带记忆增强的强化学习问题，目标是提升多步攻击链的规划与适应能力。
- **风险**：攻击不再是单次提示绕过，而是跨轮次、跨工具、跨状态的策略演化；传统只检查输入输出的网关难以发现。
- **防守动作**：记录完整工具轨迹、限制工具组合、检测异常参数序列、对跨轮状态进行污点标记。
- **来源**：https://arxiv.org/abs/2605.24274

### APT-Agent：LLM 自动化渗透测试继续推进

- **时间**：arXiv recent，发布时间待核实。
- **要点**：APT-Agent 关注使用大型语言模型执行自动化渗透测试流程。
- **风险**：公开漏洞信息、扫描结果与系统 banner 可被模型组织成更连贯的攻击路径。
- **防守动作**：在漏洞管理中增加“模型可复现性”字段；对扫描器、资产系统和代码仓库接入 agent 的行为设置审批。
- **来源**：https://arxiv.org/abs/2605.24949

## RAG、提示注入与数据投毒

### Poisoning the Watchtower：提示注入与投毒攻击融合

- **时间**：arXiv recent，发布时间待核实。
- **要点**：研究提示注入与投毒如何影响 LLM 生态中的防护或监测组件。
- **风险**：防御系统自身可能被恶意文档、日志、网页或知识库条目污染，导致误判、漏报或执行错误操作。
- **防守动作**：对进入 RAG / SIEM / 知识库的外部文本做来源标记、信任分级、去指令化处理和结果交叉验证。
- **来源**：https://arxiv.org/list/cs.CR/recent

### Five Queries Are Enough：RAG 隐私推断风险

- **时间**：arXiv recent，发布时间待核实。
- **要点**：研究少量查询下对 RAG 系统私有数据进行属性推断的可能性。
- **风险**：即使模型不直接泄露原文，攻击者也可能通过多轮查询推断知识库属性、成员关系或敏感类别。
- **防守动作**：限制查询频率、引入差分隐私式噪声或阈值、对相似问题聚类、对异常探测式查询告警。
- **来源**：https://arxiv.org/list/cs.CR/recent

## 评测、合规与隐私

### CyberMaskQA：隐私感知安全问答基准

- **时间**：arXiv recent，发布时间待核实。
- **要点**：面向网络安全领域构建隐私感知问答基准，用于评估模型在安全知识与隐私保护之间的平衡。
- **价值**：有助于衡量模型是否在回答安全问题时暴露个人、组织或敏感环境信息。
- **来源**：https://arxiv.org/list/cs.CR/recent

### TTPrint：证据驱动的 TTP 提取

- **时间**：arXiv recent，发布时间待核实。
- **要点**：面向威胁情报中的 TTP 抽取，强调证据支撑。
- **价值**：可用于 SOC 自动化威胁情报处理，但需要保留证据链，避免模型幻觉生成不存在的攻击技术映射。
- **来源**：https://arxiv.org/list/cs.CR/recent

## Agent 平台治理

### MCP 与工具调用边界

- **技术核对**：MCP 规范将工具、资源、提示、授权和 elicitation 作为协议能力；官方文档强调 MCP 可带来数据访问和代码执行能力，工具调用需用户授权、隐私控制和安全审计。
- **安全含义**：MCP server 不应默认可信；工具描述、返回内容、OAuth 流程、资源读取和外部 API 授权都需要准入与审计。
- **来源**：https://github.com/modelcontextprotocol/modelcontextprotocol

### OpenAI Agents SDK 审计信号

- **技术核对**：OpenAI Agents SDK 文档包含工具、guardrails、handoff、sessions、tracing、人工审批等能力；工具开始、工具结束和 handoff 事件可作为运行时审计信号。
- **安全含义**：企业代理平台应保留 trace，而不是只保存最终输出；高风险工具调用应支持人工审批、暂停、恢复与拒绝。
- **来源**：https://github.com/openai/openai-agents-python
