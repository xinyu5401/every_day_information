# 2026-06-13 大模型技术动态

## 1. WeaveBench 发布混合界面长程评测，当前最强组合通过率仅 41.2%

**入选理由**  
论文在过去几天进入 Hugging Face/Daily Papers 传播面，并且直接对当前最热的 computer-use agent 给出更接近生产环境的评测框架，技术信号强于常规榜单涨跌。

**技术细节**  
WeaveBench 设计了 114 个真实任务，覆盖 8 个工作域，要求模型在同一轨迹里交织 GUI 观察与点击、CLI 执行、代码修改和外部工具调用。评测运行在真实 Ubuntu 桌面与 CLI-agent runtime 中，不再把“看屏幕”“敲命令”“改代码”拆成孤立能力。论文还给出 trajectory-aware judge，会同时检查交付物、文件、截图、日志和 action trace，并显式识别伪造截图、硬编码指标等 shortcut 行为。公开结果里，最优模型-运行时组合通过率仅 41.2%，说明长程跨界面编排仍远未饱和。

**影响与处置**  
这类评测会直接影响代码 Agent、桌面 Agent、GUI 自动化产品的能力声明方式。做 Agent 基建或评测的团队应补上轨迹级审计字段，而不是只看终态文件是否存在。

**信息来源**  
- arXiv: https://arxiv.org/abs/2606.09426  
- 数据集页: https://huggingface.co/datasets/wanlilll/WeaveBench  
- 项目页: https://weavebench.github.io/

## 2. Pythagoras-Prover 用 ALF 扩增与 8k 上下文过滤，把 4B 形式化证明模型推到接近大模型区间

**入选理由**  
这是过去 24 小时内仍在扩散的高质量 arXiv 新作，且包含明确可复现的训练与评测设计，不是单纯“更大模型更高分”的常规结果。

**技术细节**  
Pythagoras-Prover 是面向 Lean 的开源证明器家族，覆盖 4B、32B 自回归模型，以及一个 4B diffusion 式证明器。方法上有三点值得跟踪：一是将已验证语料按 easy/medium/hard 分层做 curriculum SFT；二是用动态 proof-reasoning filtering，把样本压进 8k token 上下文预算；三是提出 ALF（Augmented Lean Formalisation），通过保持形式结构不变的 statement mutation 与 self-distillation 扩大稀缺的 verified corpus。结果上，4B 模型在 MiniF2F-Test 的 pass@32 达到 86.1%，超过 DeepSeek-Prover-V2-671B 的 82.4%；32B 达到 93.0%，并解出 93/672 个 PutnamBench 题目。

**影响与处置**  
该工作表明“小模型 + 高质量形式化数据工程”仍有明显上升空间。做 theorem proving、代码验证、数学推理蒸馏的团队，可以重点复核 ALF 是否能迁移到 Coq/Isabelle 或代码修复场景。

**信息来源**  
- arXiv: https://arxiv.org/abs/2606.12594

## 3. EurekAgent 把“环境工程”前置为自主科研 Agent 的核心变量

**入选理由**  
该论文今天继续在 Hugging Face Daily Papers 获得可见热度，且方法论上不是再造工作流，而是把权限、预算、产物与人类监督显式工程化，适合纳入 Agent 基建观察名单。

**技术细节**  
论文认为，自主科研 Agent 的瓶颈正在从 workflow 设计转向 environment design。EurekAgent 将环境工程拆成四个维度：`permissions engineering` 用于边界化执行与隔离评测，`artifact engineering` 用文件系统与 Git 共享中间产物，`budget engineering` 管理探索成本，`human-in-the-loop engineering` 让人工干预变成低摩擦默认能力。论文目标不是单次问答，而是围绕可优化指标做“提出方案-运行实验-复盘迭代”的闭环。

**影响与处置**  
这条路线会抬高 Agent 平台层的重要性，尤其是权限模型、审计日志、预算配额和回滚能力。对企业内部科研/代码 Agent 来说，环境工程比继续堆提示词更接近中短期落地瓶颈。

**信息来源**  
- arXiv: https://arxiv.org/abs/2606.13662  
- Hugging Face Daily Papers: https://huggingface.co/papers
