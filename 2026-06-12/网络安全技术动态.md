# 网络安全技术动态（2026-06-12）

## 1. Red Hat `@redhat-cloud-services` npm 命名空间投毒事件继续发酵，攻击链指向 OIDC 发布链与孤儿提交绕审查

**入选理由**  
该事件虽非今日初发，但在过去 24 小时内仍是开发者工具供应链方向最值得持续跟踪的高价值样本：攻击面覆盖 npm 包发布、GitHub Actions OIDC、源码仓库提交链和开发者终端凭据窃取，且已被多家一手研究复核。

**技术细节**  
Wiz 公开披露显示，`@redhat-cloud-services` 命名空间下至少 32 个包版本被植入未出现在对应源码仓库中的恶意修改，受影响包合计周下载量约 8 万。恶意版本通过新增 `preinstall` 脚本或后续 `binding.gyp` 机制，在安装阶段自动执行恶意 `index.js`。  
攻击者利用被入侵的 Red Hat 员工 GitHub 账号，向 `RedHatInsights/frontend-components`、`RedHatInsights/javascript-clients`、`RedHatInsights/platform-frontend-ai-toolkit` 等仓库推送恶意孤儿提交，绕过常规代码审查；随后以最小化工作流申请 npm 发布所需的 OIDC 身份令牌，工作流包含 `id-token: write` 权限，并对任意分支 `push` 触发。  
恶意载荷被归因为 Mini Shai-Hulud/Miasma 变种，除了窃取开发者凭据与 CI/CD 秘密外，还新增面向 GCP 与 Azure 身份的收集器，说明攻击目标已从通用 secrets 扩展到云身份与后续横向移动。

**影响与处置**  
对使用受影响包、私有镜像缓存、锁文件固化旧版本或未清理构建缓存的环境，风险并不会因 npm 下架而自动消失。短期应执行四类动作：  
1. 全量排查 `@redhat-cloud-services` 相关版本、锁文件、私有代理仓与构建缓存。  
2. 轮换开发者主机与 CI/CD 中可能暴露的 GitHub token、SSH key、云访问密钥与制品仓凭据。  
3. 审计 GitHub Actions 中 `id-token: write` 的实际使用点，核查是否存在“任意分支触发发布”与孤儿提交异常。  
4. 对 npm 发布链补充 provenance/attestation、依赖白名单与安装时脚本监控，避免将其当作普通恶意包事件低估。

- https://www.wiz.io/boards/threat-center/wiz-adv-2026-096

## 2. Jenkins 2026-06-10 安全公告修复多项核心漏洞，包含反序列化与开放重定向问题

**入选理由**  
这是过去 24 小时内可直接核验的一手官方安全公告，影响 Jenkins Core/LTS 主干版本；虽然当前未见官方在野利用声明，但其中反序列化与 URL 校验缺陷对互联网暴露实例和依赖自动化接口的环境具有现实风险。

**技术细节**  
Jenkins 官方于 2026-06-10 发布安全公告，修复多项 core 漏洞。较值得优先关注的条目包括：  
- `CVE-2026-53435` / `SECURITY-3707`：Jenkins 2.567 及更早、LTS 2.555.2 及更早版本中，攻击者可通过可控 `config.xml` 提交诱导 Jenkins 反序列化 core 或插件中的任意类型，并在后续参与 HTTP 请求处理。  
- `CVE-2026-53436` 与 `CVE-2026-53437` / `SECURITY-3711+3755`：URL 校验先于 servlet 容器路径折叠或未正确处理制表符/换行，导致协议相对 URL 绕过检查，可形成开放重定向到攻击者域名。  
- `CVE-2026-53441` / `SECURITY-3731`：节点离线原因描述在默认 UI 中被按 HTML 渲染，结合 `POST config.xml` API 可触发存储型 XSS。  
官方修复版本为 Jenkins 2.568 和 LTS 2.555.3。

**影响与处置**  
对暴露管理界面、依赖脚本化配置导入、或允许低权限用户触达部分管理接口的 Jenkins 实例，应优先升级到 2.568 或 LTS 2.555.3。短期无法升级时，至少应限制 `config.xml` 相关接口暴露、收紧反向代理访问范围、并监控异常重定向与节点离线描述字段的恶意 HTML 注入。

**信息来源**  
- https://www.jenkins.io/security/advisory/2026-06-10/
