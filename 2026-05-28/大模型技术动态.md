## 1. MiniMax-M2：低激活参数 MoE 面向真实 Agent 任务优化

- 信息来源：
  - Hugging Face 论文页: https://huggingface.co/papers/2605.26494
  - arXiv: https://arxiv.org/abs/2605.26494
- 技术细节：MiniMax-M2 系列采用 Mixture-of-Experts 架构，旗舰模型总参数约 229.9B、每 token 激活约 9.8B。论文重点强调 agent-driven data pipeline、可验证轨迹、agent-native RL、长程轨迹调度、prefix-tree merging 与训练/推理/Agent 解耦。
- 影响判断：MoE 的“低激活参数 + 长程 Agent 轨迹训练”成为提升真实工作流能力的主线之一。该方向将模型能力从静态问答推向可执行任务、深度搜索与办公协作。
- 状态：动态榜单待核实。

## 2. MobileMoE：Meta 推进端侧 MoE 语言模型

- 信息来源：
  - Hugging Face 论文页: https://huggingface.co/papers/2605.27358
  - arXiv: https://arxiv.org/abs/2605.27358
- 技术细节：MobileMoE 针对端侧部署探索 sub-billion active 参数的 MoE 语言模型，提出端侧 MoE scaling law，在移动设备内存与算力约束下寻找稀疏度、细粒度专家与 shared experts 的平衡点。论文称在同等 INT4 权重内存下，相比 dense baseline 提供更快 prefill 与 decode。
- 影响判断：端侧 LLM 不再只依赖小型 dense 模型，MoE 正在向手机等受限设备迁移。后续需要关注运行时专家调度、量化训练和端侧隐私计算的组合。
- 状态：动态榜单待核实。
