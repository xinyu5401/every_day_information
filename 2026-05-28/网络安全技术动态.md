## 1. Jenkins 发布 2026-05-27 安全公告，多个插件存在 RCE、任意文件读取、路径穿越与 XSS 风险

- 信息来源：
  - Jenkins Security Advisory 2026-05-27: https://www.jenkins.io/security/advisory/2026-05-27/
- 技术细节：公告涉及 LDAP Plugin、Active Directory Plugin、Email Extension Plugin、Pipeline: Groovy Libraries Plugin、Credentials Binding Plugin、AppSpider Plugin、Bitbucket OAuth Plugin、GitHub Integration Plugin、Job Import Plugin、Multijob Plugin、buildgraph-view Plugin 等组件。
- 主要风险：
  - LDAP/Active Directory 插件在特定 referral 场景下可能触发 SSRF 与反序列化，若 classpath 存在可利用 gadget，可能导致 Jenkins controller RCE。
  - Email Extension Plugin 可通过 `file:` URL 内联图片读取 controller 文件。
  - Pipeline: Groovy Libraries Plugin 未禁止共享库中的符号链接，可能导致文件读取。
  - Credentials Binding Plugin 对文件名清洗不足，低权限用户在特定配置下可写入任意路径并可能发展为 RCE。
  - buildgraph-view Plugin 存储型 XSS 截至公告发布时无修复版本。
- 修复建议：升级公告列出的 fixed versions；无法立即升级 Active Directory Plugin 时，可按公告设置 `hudson.plugins.active_directory.referral.ignore=true` 缓解 referral 风险；对 buildgraph-view Plugin 评估禁用或限制可配置 job/view 的用户范围。
- 影响判断：Jenkins 是 CI/CD 高权限基础设施，插件漏洞往往可串联凭据泄露、构建脚本执行和供应链投毒，应优先纳入补丁与暴露面排查。
