# 2026-05-30 网络安全技术动态

## PAN-OS GlobalProtect CVE-2026-0257 已出现利用，Rapid7 建议按关键漏洞处理

**热度判断：**Rapid7 2026-05-29 披露其 MDR 已在多个客户环境观察到 CVE-2026-0257 成功利用，最早可追溯到 2026-05-17，并称该漏洞已在 2026-05-29 加入 CISA KEV。虽然厂商 CVSSv4 标为 medium，Rapid7 基于边界 VPN 认证绕过影响建议按 critical 优先级处置。

**技术细节：**CVE-2026-0257 影响 PAN-OS 与 Prisma Access 的 GlobalProtect 场景，触发条件包括启用 authentication override，且认证覆盖 cookie 的加解密证书与其他功能复用。攻击者可利用可获取的公钥伪造 authentication override cookie，GlobalProtect gateway/portal 在服务端解密后信任 cookie 内容，从而绕过认证并建立 VPN 连接。Rapid7 披露的 PoC 会获取 HTTPS 服务证书链，逐一用证书公钥伪造 cookie 并测试 portal/gateway 是否接受。

**受影响版本与处置：**Rapid7 列出的受影响范围包括 PAN-OS 12.1、11.2、11.1、10.2 的多个低于修复版本分支，以及 Prisma Access 11.2.0、10.2.0 的部分版本。建议立即升级至 Palo Alto Networks 公告中的修复版本；若短期无法升级，应禁用 authentication override，或为该功能生成专用证书，避免与 HTTPS 或其他功能复用。检测侧重点包括 GlobalProtect Cookie authentication、local admin、异常 MAC、Vultr/Dromatics Systems 等来源、默认主机名和非人类身份登录。

- Rapid7：Observed Exploitation of PAN-OS GlobalProtect Authentication Bypass Vulnerability (CVE-2026-0257)，https://www.rapid7.com/blog/post/etr-rapid7-observed-exploitation-of-pan-os-globalprotect-authentication-bypass-vulnerability-cve-2026-0257/
- Palo Alto Networks advisory，https://security.paloaltonetworks.com/CVE-2026-0257

## Jenkins 2026-05-27 安全公告：LDAP/AD Referral RCE、凭据绑定路径穿越和多插件任意文件读

**热度判断：**Jenkins 2026-05-27 公告披露 11 个插件相关漏洞，包含 RCE、任意文件读、路径穿越、CSRF、开放重定向和权限缺失。发布时间略早于过去 24 小时，但处于本次 36 小时补漏窗口附近，且 CI/CD 控制面影响明确，保留为补漏条目。

**关键漏洞：**

- LDAP Plugin `807.v7d7de30930cf` 及更早版本：CVE-2026-48916/48917。插件会跟随 LDAP referrals，可能被转向 RMI URL 并反序列化攻击者控制数据；当 classpath 中存在 gadget 时可在 Jenkins controller 上 RCE。修复版本 `807.809.vd3a_4e5e4ec98` 不再跟随 LDAP referrals。
- Active Directory Plugin `2.41` 及更早版本：CVE-2026-48918/48919。默认跟随 AD referrals，触发路径与 LDAP 插件类似。修复版本 `2.41.1` 默认不再跟随 referrals；无法升级时可设置 `hudson.plugins.active_directory.referral.ignore=true` 缓解。
- Email Extension Plugin `1933.v45cec755423f` 及更早版本：CVE-2026-48920。`data-inline` 图片内联功能未限制 URL，可通过 `file:` URL 读取 controller 文件。修复版本 `1933.1935.v276319e3cc47` 移除该功能。
- Pipeline: Groovy Libraries Plugin `797.v90ea_a_9b_e45a_0` 及更早版本：CVE-2026-48921。共享库未禁止符号链接，控制库内容的攻击者可读 controller 文件。修复版本 `798.v5cc688825312` 禁止共享库符号链接。
- Credentials Binding Plugin `720.v3f6decef43ea_` 及更早版本：CVE-2026-48922。file/zip credential 文件名清理不足，可路径穿越写入节点任意位置；若低权限用户可配置在 built-in node 上运行的凭据，可能升级为 RCE。修复版本 `725.ve52b_2328a_fde` 改进文件名清理。

**影响与处置：**Jenkins 管理员应优先升级 LDAP、Active Directory、Credentials Binding、Email Extension 和 Pipeline: Groovy Libraries 插件；审计 controller 是否启用 LDAP/AD、共享库来源、低权限用户是否能配置文件凭据、邮件模板是否可被项目用户控制；短期内限制 Overall/Read 用户可触达的表单校验接口与 job 配置权限。

**信息来源：**

- Jenkins Security Advisory 2026-05-27，https://www.jenkins.io/security/advisory/2026-05-27/

## Metasploit 6.4.135：DirtyFrag Linux LPE、CitrixBleed 类扫描与多项 RCE 模块进入工具链

**热度判断：**Rapid7 2026-05-29 Metasploit 周报新增 5 个模块，其中 DirtyFrag Linux 本地提权和 Citrix ADC NetScaler CVE-2026-3055 扫描对攻防演练和暴露面排查有直接价值。

**技术细节：**新增模块包括 `scanner/http/citrix_netscaler_cve_2026_3055`，用于检测 Citrix NetScaler 在 SAML IdP 配置下的信息泄露，攻击形态类似 CitrixBleed，可泄漏内存并可能发现 session cookies；`linux/local/cve_2026_43284_dirty_frag` 和对应 CVE-2026-43500 模块覆盖 DirtyFrag，两者分别涉及 xfrm/ESP fragmentation path 的 page-cache write 与 RxRPC/rxkad subsystem 的 page-cache corruption；`multi/http/dompdf_rce_cve_2022_28368` 覆盖 dompdf 远程字体缓存导致的未认证 RCE；`multi/http/wp_plugin_supsystic_contact_form_rce` 覆盖 Contact Form by Supsystic 1.7.36 及更早版本 SSTI RCE。

**影响与处置：**防守方应将新增 Metasploit 模块视为利用可达性上升信号，对 Citrix ADC、Linux kernel、dompdf 和 WordPress 插件资产做版本核查；对可疑本地提权链，优先检查内核补丁、容器逃逸面、低权限 Webshell 后续提权路径和 EDR 对 page-cache 异常写入的覆盖。

**信息来源：**

- Rapid7：Metasploit Wrap Up 05/29/2026，https://www.rapid7.com/blog/post/pt-metasploit-wrap-up-05-29-2026/
